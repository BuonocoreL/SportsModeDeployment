2016 MVP AMINO SETUP DOCUMENTATION

Hardware requirements (Specific model/version #’s/vendors to come)

  • 3x Ethernet cable
  • 1x Ethernet switch
  • Google TV
  • TV
  • Amino
  • Power strip/cords implied
  • Computer/laptop

Note: Always connect the Ethernet cords before plugging in the power cords ** **Need the 5 volt power supply – confirm 5V power supply vs. 9V power supply (they look identical).

For New Amino’s only

When plugging in Amino’s fresh out of the box, no software is loaded. Upon plugging it into the network after an MDN/CORP IP is acquired, the Amino will automatically download Patrick Madigan’s version of the software. This will need to be updated to the ESPN Innovation of software for MVP deployment.

To install ESPN Innovation software

Hook up the GTV box, the Amino and the TV to the HDMI, Ethernet and power supplies appropriately. Turn on the GTV, the TV and the Amino.

  • Type: holmes.corp.espn.pvt/manageamino.php into a browser window (pc or mac)
  • Enter the IP address of the Amino
  • Select “MDN Image” in the Action box
  • Click Go
  • This will image the Amino with the Innovation image to support the MVP application.
  • If successful, you will see an ESPN Innovation logo and upgrade successful message on the TV
  • To confirm the image - hit the info button on the Amino remote control when you see the logo. A screen with Serial No., IP address, Mac Address – and confirmation of the MDN deployment build on the device should display. (The ‘I’ key brings up the info screen, ‘X’ makes it go away).
  • Label the device with version: MDN101 (hand written on tape affixed to the Amino)
  • To get out of the info screen on the Amino, use the home button on the google remote to bring up the “Use X app to start” box. Select ESPN app from the list.

Pairing the Amino device on the 2016 GTV app

  • Activate the GTV app
  • Open the menu in the app using the menu key on the GTV remote and select “Settings”.
  • Select Advanced preference
  • PW “12345” in the PW field
  • Select Pairing settings
  • Select ‘Pairing’ under Device pairing settings
  • Select Pairing device and chose the Amino option
  • Select Pairing device identifier and enter the serial number (case sensitive) from the label on the Amino Device
  • ‘OK’
  • Back out of Advanced preference menu and confirm the GTV remote can be used to tune the TV.

To get the IP address on the Amino – if it doesn’t come up on the screen after plugging the Amino in

  • Go into Infoblox page (web browser) https://infoblox
  • Enter your corp credentials into the UN and PW fields to login
  • Use the search feature to enter the mac address of the Amino (with colons) and click search. This should return the IP address for the box.

To check the version on the Amino, plug the Amino in, click the info button, determine the build on the Amino.

User will not need the Amino remote; all control is managed with the GTV remote.
